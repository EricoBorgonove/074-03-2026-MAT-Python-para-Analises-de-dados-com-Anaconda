Nosso objetivo é que, ao final da jornada, você seja capaz de:

- 🐍 Utilizar **Python** como ferramenta para análise de dados  
- 📊 Trabalhar com **datasets reais**  
- 🧠 Aplicar **técnicas básicas de análise exploratória de dados**

# 🎯 Objetivos do Curso

Durante o curso você irá aprender a:

- Compreender o funcionamento do **ecossistema científico do Python**
- Instalar e configurar corretamente o **Anaconda**
- Criar e gerenciar **ambientes virtuais**
- Utilizar ferramentas como:

  - **Jupyter Notebook**
  - **Pandas**
  - **NumPy**
  - **Matplotlib**
  - **Seaborn**

- Realizar:

  - limpeza de dados
  - transformação de dados
  - análise exploratória
  - geração de gráficos
  - interpretação de resultados
